# ⚙️ Rust Dedicated Server: Установочный скрипт

![Ubuntu](https://img.shields.io/badge/Ubuntu-22.04%20%2F%2024.04-E95420?style=for-the-badge&logo=ubuntu&logoColor=white)
![Rust Server](https://img.shields.io/badge/Rust_Server-CE422B?style=for-the-badge&logo=steam&logoColor=white)
![SteamCMD](https://img.shields.io/badge/SteamCMD-171A21-000000?style=for-the-badge&logo=steam&logoColor=white)

Скрипт для автоматизированного развертывания выделенного сервера Rust на Ubuntu 22.04 / 24.04. Выполняет установку зависимостей, загрузку файлов через SteamCMD, базовую настройку сети и создание системной службы.

## ✨ Особенности

* ⚡ **Развертывание одной командой.**
* 🎛️ **Интерактивная конфигурация** основных параметров при установке (название сервера, пароль RCON, seed, размер карты, лимит игроков).
* ⚙️ **Управление через systemd**: сервер работает как фоновая служба с автоматическим перезапуском при сбоях и стартом при загрузке ОС.
* 🛡️ **Настройка сети**: автоматическое добавление правил UFW для необходимых портов.
* 💻 **Встроена утилита rcon-cli** для локального администрирования через консоль.
* 📄 Конфигурация вынесена в единый файл `config.env`.
* 📱 Поддержка приложения Rust+.
* 🛠️ Набор bash-скриптов для обслуживания (обновление, вайп, рестарт по расписанию).

---

## 🚀 Установка

Выполните в терминале сервера следующую команду:

```bash
curl -fsSL "[https://raw.githubusercontent.com/apposumlive/rust-server-installer/main/install.sh](https://raw.githubusercontent.com/apposumlive/rust-server-installer/main/install.sh)" -o install.sh && bash install.sh && rm install.sh
```

> ⚠️ **Примечание:** Загрузка файлов через SteamCMD занимает от 2 до 5 минут в зависимости от пропускной способности сети. Не прерывайте процесс сочетанием `Ctrl+C`. Скрипт выведет уведомление об успешном завершении установки.

## 🔍 Проверка статуса

После установки сервер автоматически запустится. Для проверки состояния используйте следующие команды:

1. 🟢 **Проверка системной службы:**
   ```bash
   systemctl status rustserver
   ```
   *(Ожидаемый статус: `active (running)`)*

2. 📜 **Просмотр логов запуска в реальном времени:**
   ```bash
   tail -f ~/rustserver/logs/latest.log
   ```

## 🕹️ Управление сервером

Для базового администрирования предусмотрены готовые скрипты и стандартные системные команды:

| Действие | Команда |
| :--- | :--- |
| **Перезапуск** 🔄 | `sudo systemctl restart rustserver` |
| **Остановка** 🛑 | `sudo systemctl stop rustserver` |
| **Обновление** ⬇️ | `~/rustserver/scripts/update.sh` |
| **Смена seed (Вайп)** 💥 | `~/rustserver/scripts/wipe.sh` |
| **Сообщение в чат** 💬 | `~/rustserver/scripts/rcon.sh say "Текст сообщения"` |

---

## 📝 Настройка конфигурации

Основные переменные хранятся в файле конфигурации. Для их изменения:

1. Откройте файл в редакторе:
   ```bash
   nano ~/rustserver/config.env
   ```
2. Внесите изменения (например, измените название сервера или RCON пароль), сохраните файл и перезапустите службу:
   ```bash
   sudo systemctl restart rustserver
   ```

## ⏱️ Автоматический рестарт (cron)

В комплекте поставляется скрипт `scheduled_restart.sh`, который отправляет уведомление в игровой чат за 10 минут до фактической перезагрузки.

Для настройки ежедневного рестарта (например, в 09:00), добавьте задачу в планировщик:

1. Откройте crontab:
   ```bash
   crontab -e
   ```
2. Добавьте следующее правило:
   ```cron
   50 8 * * * /home/$(whoami)/rustserver/scripts/scheduled_restart.sh
   ```
